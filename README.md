# car_price_analysis
Analyzed used car data to identify key factors affecting resale prices using Python, SQL, and Power BI
##  Car Price Analysis & Insights Dashboard

This project is an end-to-end Data Analyst workflow that analyzes used car data to uncover key factors affecting resale prices. The analysis includes data cleaning, exploratory data analysis (EDA), SQL-based querying, and an interactive Power BI dashboard.

The dataset was preprocessed using Python (Pandas, NumPy) to handle data type inconsistencies and prepare features such as mileage, engine, and power. The cleaned data was then stored in PostgreSQL, where analytical queries were performed to extract business insights.

Finally, a Power BI dashboard was built to visualize trends such as price variation by brand, fuel type, transmission, vehicle age, and usage (km driven). The dashboard includes KPI cards, interactive filters, and multiple visualizations to support data-driven decision making.

###  Tools & Technologies

* Python (Pandas, NumPy, Matplotlib, Seaborn)
* PostgreSQL (pgAdmin4)
* Power BI
* Excel / CSV

###  Key Insights

* Vehicle age and km driven negatively impact resale price
* Automatic cars and diesel vehicles tend to have higher resale value
* Premium brands dominate the high-price segment
* Engine power and performance strongly influence pricing

###  Outcome

Built a complete data analysis pipeline and an interactive dashboard to provide actionable insights into used car pricing trends.
